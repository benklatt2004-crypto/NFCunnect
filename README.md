NFCunnect ist eine Juniorfirma von uns. 
Sie erstellt Kupferschlüsselanhänger mit beklebtem, individuell programmierbarem NFC-Chip. 
Die Unternehmensfarben sind der CI zu entnehmen.
Fokus beim Programmierung auf Neuwertigkeit, Bewegung und Interaktion legen.

---

# Website

Statische Website der Juniorfirma NFCunnect – läuft direkt auf **GitHub Pages**.

**Live:** https://benklatt2004-crypto.github.io/NFCunnect/

## Funktionen

- Hero mit animiertem Logo (schwebend, rotierende Ringe)
- Produktsektion mit 3D-Hover-Effekt
- Galerie mit 3D-Flip-Karten (Hover = Karte umdrehen)
- Über uns mit animiertem Zähler
- Kontaktformular → öffnet E-Mail-Programm an `nfcunnect@outlook.com`

## Lokaler Test

Einfach `index.html` im Browser öffnen – kein Server nötig.
